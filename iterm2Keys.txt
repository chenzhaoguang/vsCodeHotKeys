1、编辑
清除当前行：control + U
删除到文本末尾：control + k
交换光标处文本：control + t
清屏1：command + r 或者 command + k
清屏2：control + l (保留用户信息)

2、展示
搜索所有tab:command + option + E
高亮当前鼠标的位置:command + /
快速唤出:command + .
当前行高亮:option + command + ;
显示时间:command + shift + E
快速打开: command + shift + O
当前tab充满宽度:command + shift + 回车
命令回放:option + command + B
关闭pane tab : option + command + w
关闭treminal : command + shift + w


3、历史
弹出历史命令窗口:command + shift + h
弹出自动完成:command + ;
历史命令回放:command + option + B
